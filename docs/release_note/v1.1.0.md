# v1.1.0

リリース日: 2026-03-23

## ハイライト
- 時系列マージ時の間引き機能を追加
- 末尾ステップを必ず採用するかどうかを選択可能に変更
- ユーザー向け更新スクリプトを追加

## 追加・変更内容
- 詳細設定に `間引き設定` を追加
  - `間引きしない`
  - `等間隔で間引く`
- `間引き間隔（Nステップごと）` を追加
- `末尾ステップを必ず採用` を追加（ON/OFF）
- 間引き後の採用件数を実行ログに表示
- 既存インストール更新用の `update_solver.bat` を追加
  - 既存 `CgnTM` をバックアップしてから新規ファイルを配置
  - `venv` / `__pycache__` / `*.pyc` はコピー対象外

## ダウンロード
- このリリースページを開いた状態で、**Assets** から配布物をダウンロードしてください
