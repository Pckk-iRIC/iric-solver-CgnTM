# CgnTM ユーザーマニュアル

このドキュメントは、iRIC v4 の分割出力（Solution*.cgn）を統合するソルバー「CgnTM v4」のユーザー向けマニュアルです。

## できること
- 分割出力された CGNS（Solution*.cgn）をまとめ、Case1.cgn 相当の時系列結果を作成する。
- 入力は `.ipro` / プロジェクトフォルダ / result フォルダに対応。
- 出力名は `Case1.cgn` または任意の別名を指定できる。

## ドキュメント構成
- 使い方: [usage.md](usage.md)
- ダウンロードと配置: [install.md](install.md)

## iRIC環境の注意点
- iRICのインストール時に Miniconda3 もインストールする必要があります。
- Miniconda3 の存在確認は `C:\Users\<username>\iRIC_v4\Miniconda3` を参照してください。  
  ※ `<username>` は各ユーザー名に置き換えてください。
